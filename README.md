---

Logic Gates Implemented

AND Gate

Output becomes HIGH only when both inputs are HIGH.

OR Gate

Output becomes HIGH when at least one input is HIGH.

NOT Gate

Produces inverted output.

XOR Gate

Output becomes HIGH when inputs are different.

---

Half Adder

The Half Adder is a combinational circuit used to add two binary bits.

Outputs

- SUM = XOR of inputs
- CARRY = AND of inputs
